Description List, Special Entity Characters

TCP
Transmission Control Protocol is a method(protocol) used along with the Internet Protocol(IP) to send data in the form of message unitys, called packets, between computers over the internet.
Internet Protocol is the method or protocol by which data is sent from one computer to another on the Internet. Each computer on the Internet is uniquely identified by an IP address.
FTP
File Transfer Protocol used to exchange files between computers on the Internet.
HTTP
Hypertext Transfer Protocol is the Protocol used for exchanging test, graphic images, sound, video, and other multimedia files on the web.
